# carobiner

R package with functions that supports standardizing agricultural research data with the [terminag](https://controvoc.github.io/terminag/) vocabulary. The package is used by the [Carob](https://github.com/carob-data/carob/) project, but you can also use its functions in the context of your own standardization efforts.

You need to install this package if you want to contribute a script to Carob, or if you want to improve an existing script.

You can install the package with: 

```
remotes::install_github("carob-data/carobiner")
```
